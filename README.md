# 🎓 Application Web – Centre de Formation Continue (CFC)

## 📌 Description du projet

Ce projet consiste à modéliser et développer une application web pour la gestion des formations d’un Centre de Formation Continue (CFC).

L'application permet aux candidats de consulter les formations disponibles, de se préinscrire et de soumettre un dossier d'inscription.  
Les administrateurs et coordinateurs peuvent gérer les formations et valider les candidatures.

---

# 👥 Acteurs du système

Le système comprend les acteurs suivants :

### Super Admin
- Gestion des établissements
- Gestion des administrateurs
- Paramètres globaux du système

### Admin d’établissement
- Gestion des formations de son établissement
- Validation ou refus des dossiers des candidats

### Coordinateur
- Gestion d'une formation
- Ouverture et fermeture des inscriptions

### Candidat
- Consultation du catalogue
- Préinscription
- Soumission du dossier
